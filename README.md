### Smooth Scroll
For smooth scrolling i use just some basic jQuery. 
```javascript
$(document).on('click', 'a[href^="#"]', function (event) {
  event.preventDefault();

  $('html, body').animate({
    scrollTop: $($.attr(this, 'href')).offset().top
  }, 1500);
});
```
